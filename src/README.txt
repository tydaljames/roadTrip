-- RoadTrip --
Created by Tyler D'Alessandro - 30 NOV 2020

This program takes user inputs and computes the shortest TripSegment between cities and attractions.

How to use:
1) Enter starting city.
2) Enter ending city.
3) Enter # of attractions to visit on the way.
4) Enter each attraction one by one.


The program will find the shortest TripSegment to visit all attractions and reach your City. Enjoy!
